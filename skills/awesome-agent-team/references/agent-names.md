# Agent Name Pool / Agent 名字池

> **Usage**: This file is a human-readable reference. The machine-readable version is at `assets/names.json`.
>
> **用途**: 本文档供人类阅读参考。机器可读版本在 `assets/names.json`。

## Overview / 概述

This file contains 200+ carefully selected English first names used by the Awesome Agent Team plugin. When you run `./scripts/install.sh`, names are randomly selected from this pool and assigned to your team members — each installation produces a unique team.

本文档包含 200+ 个精选英文名，供 Awesome Agent Team 插件使用。运行 `./scripts/install.sh` 时，会从此池中随机选择名字分配给团队成员 — 每次安装都会产生一个独特的团队。
